## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| YAML config → CLI presentation | User-supplied selector strings from `prepare.hide` are interpolated into the warning message in printSelectorWarnings; embedded ANSI/newline characters could in theory clobber output formatting. |
| Library → CLI (silent boundary) | Phase 5 capture library invokes the onProgress callback; the callback runs in the CLI's scope. If the library throws inside onProgress, it would propagate; the CLI must trust the library not to abuse the callback. |
