**File A — `src/capture/index.js` (NEW):**

The orchestrator is the only fs-touching code in Phase 5. It composes 05-01's `captureFrames` and 05-02's `stitchFrames`, then writes the resulting Buffer via `node:fs/promises`. Three steps in fixed order:
